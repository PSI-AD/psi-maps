Map Engine & Tools

Professional-grade spatial tools built on Mapbox GL JS

MapCommandCenter.tsx

Map Command Center

Floating toolbar with five control groups. All interactions dispatched via CustomEvents so components stay decoupled.

  • Camera — Bird's Eye, 3D perspective (60° pitch), 360° orbit, Reset North
  • Style — Streets, Light, Outdoors, Satellite toggle
  • Tools — Zoom±, Crosshair focus, 3D buildings toggle, PNG map export
  • Lasso — Freehand polygon draw-to-filter, clear selection
  • Insights — ROI heatmap overlay, Time Machine launcher
TimeMachine.tsx

Time Machine

Cinematic satellite timelapse using Esri Wayback archive tiles (2014–2026).

  • Auto-plays forward/backward through historical years
  • Smooth crossfade between tile layers (50ms intervals)
  • Preloads surrounding 3×3 tile grid for seamless transitions
  • 360° camera orbit during playback
  • Speed control: 3s / 5s / 7s / 10s per frame
  • Bird's Eye or 3D view modes
  • Dot timeline for direct year selection
ROIHeatmapOverlay.tsx

ROI Investment Heatmap

Visual heat zones overlaid on the live map showing relative investment yield concentration by district.

  • Toggle via MapCommandCenter or AI Chat
  • Color-coded intensity zones (green → red)
  • Mutually exclusive with Time Machine
MapCanvas.tsx

Map Canvas Core

39KB core rendering layer wrapping react-map-gl with full event and layer management.

  • Project markers with pulse animation & clustering (use-supercluster)
  • Landmark markers by category (School, Hotel, Culture, etc.)
  • Community boundary GeoJSON overlays
  • Draw control for polygon lasso selection
  • 3D buildings layer (fill-extrusion)
  • Error Boundary with Firebase crash reporting
BeforeAfterSlider.tsx

Before / After Slider

Drag-to-compare historical vs. current satellite imagery for any project location. Touch and mouse input supported.

ViewSimulator.tsx

View Simulator

Simulates the real-world view from a property's floor — 360° panoramic approximation using map pitch and bearing controls.

AI & Intelligence Engine

Gemini-powered insights and contextual spatial analysis

AIChatAssistant.tsx

AI Chat Assistant

Context-aware action panel powered by Google Gemini. Auto-opens when a project, landmark, community, or developer is selected. Adapts suggestions to active context.

  • Project context — nearby projects, same-developer showcase, off-plan/completed filters, landmark neighbourhood tour, lead capture
  • Community context — all community projects, top developers, off-plan/ready
  • Developer context — full portfolio, by-city breakdown, off-plan/completed
  • Landmark context — nearby projects within radius, by status or developer
  • Classic card panel or modern floating bubble UI (toggle in Admin)
  • Built-in lead capture mini-form (name + phone → Firebase)
  • Auto-dismisses after 14s, pauses on hover
  • Fully suppressed during any active tour
valuationEngine.ts

Valuation Engine

Haversine-formula geo-spatial proximity calculator that auto-generates investment premiums.

  • Calculates great-circle distance to all landmarks
  • "Cultural District" multiplier for sub-500m proximity (e.g. Louvre Abu Dhabi)
  • Outputs yield %, appreciation %, and landmark premium %
  • Powers the Investment Analysis tab in every project sidebar
InvestmentStoryPanel.tsx

Investment Story Panel

Narrative-style investment timeline showing projected value growth, ROI milestones, and area development story powered by Gemini synthesis.

WalkabilityRadial.tsx

Walkability Score Radial

Animated SVG radial chart showing walkability, transit, and bike scores for any project. Calculated from POI density within 800m radius.

RentVsBuyCalculator.tsx

Rent vs Buy Calculator

Interactive financial model comparing rent and purchase scenarios. Inputs: price, mortgage rate, tenure. Outputs: break-even year, total cost comparison, monthly delta, NPV.

NoiseTrafficBadge.tsx

Noise & Traffic Badge

Visual indicator of estimated noise level and traffic density near a project, based on proximity to roads, airports, and commercial zones.

Performance & Reliability

Sub-200ms interactions, graceful offline, real user monitoring

Smart Cache Two-tier cache: sessionStorage for tab-lifetime state, IndexedDB for durable offline storage of projects and landmarks. 24h TTL with stale-while-revalidate. smartCache.ts · 25KB
Map Region Cache Saves last Mapbox viewport (lat/lng/zoom/pitch/bearing) to localStorage on every map move. Restored instantly on next app open. 7-day TTL. localPersistence.ts · saveMapRegion()
Background Sync Offline action queue for favourites, inquiries, and feedback. Replays automatically on reconnect using the Web Background Sync API. backgroundSync.ts · 14KB
Network Resilience Exponential backoff retry for Firestore reads, request deduplication, timeout handling, and connection quality detection. networkResilience.ts · 11KB
Real User Monitoring Web Vitals (LCP, FID, CLS, TTFB) reported to Firebase Analytics with good/needs-improvement/poor ratings. Fires on every page load. performanceEngine.ts · logWebVitals()
Device Tier Detection Classifies device as high/mid/low based on CPU cores, RAM, and network type. Logs device_profile to Firebase Analytics. Triggers graceful degradation (disable 3D buildings, reduce animations) on low-end devices. performanceEngine.ts · detectAndReportDeviceTier()
Skeleton UI GPU-accelerated shimmer placeholders for every loading state — project cards, sidebar panels, image slots. Prevents layout shift. SkeletonUI.tsx · 18KB

Security & Compliance

Production-hardened access controls

Firestore Rules

Data Access Control

Public collections (projects, landmarks, communities) are read-only. All writes require request.auth != null. Lead/report/error_log collections are append-only from clients. Admin collections fully auth-gated.

Android R8

Code Minification

Release builds use minifyEnabled true + shrinkResources true with the optimised ProGuard ruleset. Debug builds remain unminified.

App Check

reCAPTCHA Enterprise

initAppCheck() is wired and ready. A reCAPTCHA Enterprise site key is required to activate full enforcement in the Firebase Console.

HTTPS

Transport Security

Firebase Hosting enforces HTTPS for all traffic. API keys are base64-encoded in the bundle and restricted by domain in Google Cloud Console.

Accessibility

WCAG AA Compliance

Minimum 44×44px touch targets enforced via CSS. Focus rings on all interactive elements (:focus-visible). No user-scalable=no. Missing alt attributes flagged visually in development.

Offline Queue

Data Integrity

User actions (favourites, inquiries, feedback) queued to IndexedDB when offline. Replayed with idempotent IDs on reconnect to prevent duplicate submissions.

Roadmap

Planned and upcoming platform phases

Phase 1 — Core Platform Shipped

Interactive map, project database, AI insights, PDF brochures, admin CMS, neighbourhood intelligence, PWA, Capacitor native builds.

Phase 2 — CRM Integration In Progress

Real-time unit availability via crm-proxy-server. Live "Sold / Available" marker colours via CRM webhook stream. Developer portal for unit management.

Phase 3 — AI Comparison Engine Planned

Multi-project side-by-side analysis with Gemini-generated narrative comparison. Ranked shortlists by yield, ROI, and lifestyle score.

Phase 4 — AR Floor Plan Visualiser Planned

WebAR session (8th Wall / A-Frame) letting buyers walk through floor plan layouts in augmented reality using device camera.

Phase 5 — Isochrone & Lasso Tools Planned

Drive-time boundary overlays via Mapbox Isochrone API. Freehand polygon selection with Turf.js intersection filtering.

Phase 6 — App Store Launch Awaiting Credentials

Apple App Store (requires Apple Developer enrolment + code signing). Google Play Store (requires Keystore + Play Console account). Screenshots, metadata, privacy policy.
